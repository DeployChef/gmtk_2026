Maya settlement. Great Pyramid. Gods hungry for tribute.  
The end-of-the-world clock is already running — and nothing builds, crafts, or saves itself.

You **drag** resource cards. You **assign** villagers by hand.  
Every second is a choice: feed production, or feed the gods.

> Run the settlement. Appease the divine. Outlast the countdown.
