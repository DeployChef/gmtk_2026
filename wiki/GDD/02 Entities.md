## 4. Основные механики и сущности

### Единый формат: всё — карточки / ресурсы

Ресурсы, активки и **жители** — один тип интерактива: UI-карточки из **`ResourceDefinition`**. На экране они лежат в **лотках** (по типу), не россыпью в одной ленте. См. [[../Architecture/06 Inventory|Architecture: Inventory]].

### 4.1. Жители (карточки NPC)

**Данные:** `Resource_Villager` (`ResourceDefinition`, id `Villager`, без лимита стека).

**В лотке** лежат только **свободные** карты (available). Assigned как карты **не показываются**.

**Счётчик у лотка** (как у остальных типов, но другая семантика max):
- **available** — свободны для работы / DnD
- **total** — available + назначенные в здания

#### Назначение в здание

| Действие | Результат |
| --- | --- |
| DnD карты жителя на drop-zone здания | `+1` рабочий (если не max), карта **убирается из инвентаря** |
| Кнопка `[+]` | то же; **если available = 0 — кнопка неактивна** |
| Кнопка `[-]` | `−1` рабочий, карта **возвращается** в инвентарь / лоток |

Без рабочих производственные здания не работают. Каждый рабочий в здании = бывшая available-карта.

#### Жертвоприношение

Жителя **нельзя** отдать напрямую Пирамиде. Жертва идёт через **Алтарь**: карта жителя уничтожается как input → на выходе **Кровь**. Кровь (как и другие ресурсы) отдаётся Пирамиде. См. [[03 Economy]].

#### Появление новых жителей

Цепочки вроде «Дом Предков + еда» выдают жителя в инвентарь (`TryAdd`).

**Старт игры:** **1×** `Resource_Villager` через `GameStartState`.

### 4.2. Карточки ресурсов и активок

Каждый тип — свой лоток со **стеком** и **ёмкостью** (`TrayCapacity`). Если места нет — новая карта **сгорает** (спавн + destroy VFX).

DnD: верхняя карта из лотка → World Space drop-zone здания / Пирамиды (на Пирамиду — только ресурсы, не жители).

#### Ресурсы

Тратятся на производство других ресурсов или отдаются Пирамиде. Рецепт здания ссылается на `ResourceDefinition` (input / output).

#### Активки (Баффы)

Специальные карточки, которые можно «бросить» на здание для временного эффекта:

| Активка | Эффект |
| --- | --- |
| **Ускорение** | +100% к скорости производства на 30 сек |
| **Защита** | Иммунитет здания к следующему катаклизму |

> **Примечание:** любое здание при выдаче ресурса имеет небольшой шанс дополнительно выдать Карточку-Активку. См. [[03 Economy]].
