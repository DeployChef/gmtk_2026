## 6. Таймлайн и фазы (эры)

Вся партия делится на **фазы** (ориентир: **~8**). Каждая фаза — отрезок таймлайна с:

- **длительностью** (может отличаться: 60с / 120с / …);
- **требованиями Пирамиды** (офферы: ресурс, количество, **цена в секундах за карту**);
- **разблокировкой зданий** (`unlockBuildingIds[]`) — руины получают оффер стройки (см. [[03 Economy#5.2. Строительство (оффер на слоте)|Economy]]);
- позже — **модификаторами правил** эры (вторым заходом);
- визуалом сегмента на TopBar (цвет, тултип).

Данные фаз — **ScriptableObject-последовательность**. Баланс `+`/`−` секунд — **ручной** в настройках фазы (без автоформулы «вернуться к 99»).

### Два независимых «часа»

| Часы | Где UI | Что делает | Проигрыш? |
| --- | --- | --- | --- |
| **Таймлайн / фазы** | TopBar (сегменты-слайдеры в ряд ≈ один длинный слайдер) | Двигает эры, проверяет офферы к дедлайну фазы | Нет: провал → гнев богов, переход дальше |
| **Таймер Конца Света** | World Space над Пирамидой | Тикает вниз; верный оффер → **+цена из SO**; неверный → reject + **±penalty** | Да: `00:00` |

Стартовый запас таймера (`baselineSeconds`, напр. 99) — только точка старта. К концу фазы remaining **может плавать** выше/ниже — так создаётся стресс. Геймдизайн крутит `secondsReward` у элементов оффера и `wrongOfferingTimerDelta`.

### Оффер в данных фазы

Каждая позиция оффера:

| Поле | Смысл |
| --- | --- |
| `ResourceDefinition` | какой ресурс |
| `count` | сколько карт нужно |
| `secondsReward` | сколько секунд **прибавить** за одну сданную карту этого типа |

Пример: фаза 1 — 6× кукуруза, каждая `secondsReward = 8` (не обязательно `duration/count`).

### Подношения

- Принимаются **только** карты, ещё нужные текущему офферу → `+secondsReward` для этой позиции.
- Иначе: **reject** + изменить таймер на **`wrongOfferingTimerDelta`** (заложенное ± число в конфиге, напр. `−1`; можно выставить `0` или даже плюс для экспериментов).
- Житель на Пирамиду — нет. Кровь — когда фаза требует.

### Досрочный оффер

Оффер закрыт раньше конца сегмента → **ждём конца фазы** по таймлайну. Без skip. В конце без молнии.

### Провал оффера

Не закрыто к дедлайну → гнев (молния) → следующая фаза. Не проигрыш.

### Модификаторы эры

Вторым заходом.

### Debug

Кнопки **Jump to phase** / **Grant All Cards** — окно **They Will Descend → Cheat Panel** (только Play Mode). Jump сбрасывает здания по `CheatPanelConfig` (Built list / Locked + cumulative unlocks) и карты. Runtime UI нет.

### UI

Сегменты-слайдеры, годы на TopBar; World-таймер над placeholder-пирамидой.
